# Karat Mebli

Landing page for a furniture manufacturer. Built with React + Vite + Tailwind CSS v4.

## Stack

- React 19 + TypeScript
- Vite 6
- Tailwind CSS v4
- Framer Motion (`motion/react`)
- Lucide React

## Features

- UA/EN language switcher (Ukrainian default)
- Upholstery gallery with material specs
- B2B contact form
- Floating social links (WhatsApp, Telegram, Viber)

## Local dev

```bash
npm install
npm run dev
```

## Deploy

Pushes to `main` auto-deploy to GitHub Pages via Actions.
